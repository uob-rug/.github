# 📘 University of Birmingham (UoB) R User Group

Welcome to the University of Birmingham R User Group — a community for students, researchers, and staff who use the R programming language for data analysis, statistics, reproducible research, and data‑driven projects.
We welcome all skill levels, from complete beginners to advanced users.

For more details, see [https://github.com/uob-rug/uob-rug.github.io](https://github.com/uob-rug/uob-rug.github.io)
